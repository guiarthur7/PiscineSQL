# PiscineSQL

## 📋 Contenu

- **DDL** : Création et modification de tables (`CREATE TABLE`, `ALTER TABLE`, `RENAME TABLE`)
- **DML** : Manipulation de données (`INSERT`, `UPDATE`, `DELETE`)
- **SELECT** : Requêtes avec `WHERE`, `ORDER BY`, `LIMIT`, `DISTINCT`
- **Agrégations** : `COUNT`, `SUM`, `MIN`, `MAX`
- **JOINs** : `INNER JOIN`, `LEFT JOIN` sur plusieurs tables
- **Conditions** : `AND`, `OR`, `NULL`
